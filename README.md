# Dambou Web — dambou.fr

Code PHP du site vitrine et des pages pro de Dambou.

## Structure
- `index.php` — routing principal (détecte /[slug])
- `pro/page.php` — page vitrine du pro
- `abonnement.php` — page de souscription Stripe
- `.htaccess` — réécriture d'URL

## Déploiement
Push sur `main` → déploiement automatique sur Hostinger via Git.
